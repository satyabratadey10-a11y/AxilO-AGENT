# UI/UX Skill: Claude AI Aesthetic

## Core Principles
- **Functional Minimalism**: Every element must serve a purpose. Remove unnecessary borders and heavy decorations.
- **Paper-like Depth**: Use soft backgrounds (#F9F9F8) and very subtle box-shadows to create layers without harsh lines.
- **Typography First**: Prioritize readability. Use high-quality sans-serif fonts (Inter, Geist, or system-ui) with generous line-height (1.6).
- **Human-Centric Interaction**: Transitions should be smooth (200ms ease-in-out). Buttons should have rounded corners (8px to 12px).
- **The 'Input-Focus' Experience**: The chat input is the heart. It should be centered, spacious, and expand naturally.

## Color Palette
- Primary Background: `#FFFFFF` or `#F9F9F8`
- Secondary Background: `#F0F0EE`
- Text: `#1D1D1B` (High contrast but not pure black)
- Muted Text: `#676765`
- Accent: `#D97757` (Claude's signature clay/orange for specific highlights)

## Implementation Guidelines
1. Use CSS Grid and Flexbox for responsive, centered layouts.
2. Implement 'Skeleton Screens' for loading states.
3. Ensure mobile responsiveness with a bottom-heavy navigation/input focus.
4. Use Lucide-style thin icons for a modern, lightweight feel.
