AxilO Swarm Framework

The ultimate autonomous AI Orchestrator running natively on Termux. AxilO utilizes a specialized Swarm Protocol to spawn worker drones, manage complex local systems, and execute bash commands through a beautifully crafted universal ASCII UI.

Core Capabilities:

  • Worker Drones: Spawns specialized sub-agents to preserve main context tokens.
  • Token Conservation: Dynamically bypasses reasoning loops for trivial tasks.
  • Strict Execution Pipes: Sandboxes standard errors to prevent terminal UI bleed.
  • Universal Interface: Rock-solid ASCII geometry guaranteed to render on all Android mobile fonts.
